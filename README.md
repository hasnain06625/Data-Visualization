## E-Commerce Sales Analysis

# Project Overview
This project analyzes an e-commerce sales dataset using Python. It calculates total sales, identifies the best-selling product, and visualizes the data using charts.

# Objectives
- Load the dataset
- Analyze sales data
- Find the best-selling product
- Create charts for visualization
- Generate simple business insights

# Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

# Dataset
The dataset contains the following columns:
- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

# Features
- Load CSV file
- Calculate total sales
- Product-wise sales analysis
- Region-wise sales analysis
- Bar Chart
- Pie Chart
- Business Insights

## Output
- Total Sales
- Best Selling Product
- Sales by Product (Bar Chart)
- Sales by Region (Pie Chart)
- Business Insights
